# 🏙️ 二層都市 · 二重チェーン清算ネットワーク

**Nekop2p — The Dual-Chain Settlement Network**

> **検証可能性が信頼に代わる** — すべての操作は、オンチェーンデータ、ゼロ知識証明、または公開アルゴリズムを通じて事後監査可能です。

---

## 概要

**Nekop2p（二層都市・二重チェーン清算ネットワーク）** は、P2P 通信、デュアルチェーンアーキテクチャ（ブライトチェーン／ダークチェーン）、ゼロ知識証明（ZK）、およびカオス決済プールを統合した分散型ネットワークシステムです。

その中核的理念は、**暗号学的検証可能性**が従来の社会的信頼メカニズムに取って代わることです。Nekop2p において、ノードは信頼されず、信頼は暗号鍵の保有者間にのみ存在します。

---

## 主な特徴

| 特徴 | 説明 |
|:---|:---|
| 🔗 二層都市モデル | 明域（透明で公開された陽光の世界）と暗域（匿名で遮蔽された防空壕）が二重の経済システムを形成 |
| 🛡️ 五層防御 | E2E 暗号化 → ZK 遮蔽 → オニオンルーティング → オフライン招待 → 社会的認識 |
| 🔐 ゼロ知識証明 | 6 つの Groth16 ZK 回路（身分・信用・返済・作業証明・ドアロック・手形） |
| 🌀 カオス決済プール | 金額の 3〜7 片分割、1〜90 日のランダム化、ソース難読化 |
| 🧅 オニオンルーティング | 3 ホップ以上の中継、エンドツーエンドの身元秘匿 |
| 💱 二重チェーン清算 | ブライトチェーン（信用・保証・求償）+ ダークチェーン（匿名貸付・手形・ドアロック） |
| 🎮 ゲームアプリケーション層 | 明域上の戦略ゲームプレイ、暗域上のプライバシー決済チャネル |
| 📡 P2P 通信層 | Noise IK/NK ハンドシェイク · Signal ダブルラチェット · ChaCha20-Poly1305 フレーム暗号化 |
